📖 Commandes disponibles

Découvrez toutes les fonctionnalités de The Kraken Bot

🎶 Musique

  • [prefix]play [nom/lien] — Joue une musique
  • [prefix]playlist [lien/nom] — Joue une playlist
  • [prefix]add [nom/lien] — Ajoute une musique à la file d'attente
  • [prefix]skip — Passe à la musique suivante
  • [prefix]stop — Arrête la musique
  • [prefix]pause / [prefix]resume — Pause/Reprend la musique
  • [prefix]queue — Affiche la file d'attente
  • [prefix]volume [0-100] — Définit le volume
  • [prefix]loop — Active/Désactive la boucle
  • [prefix]nowplaying — Affiche la musique en cours

🔨 Modération / Sanction

  • [prefix]mute @user — Mute un utilisateur
  • [prefix]unmute @user — Unmute un utilisateur
  • [prefix]tempmute @user [durée] — Mute temporairement
  • [prefix]ban @user [raison] — Banni un utilisateur
  • [prefix]listwarn @user — Liste les avertissements
  • [prefix]kick @user [raison] — Expulse un utilisateur
  • [prefix]unban [userId] — Débanni un utilisateur
  • [prefix]warn @user [raison] — Avertit un utilisateur
  • [prefix]unwarn @user — Retire un avertissement

🛠️ Modération / Gestion

  • [prefix]clear [nombre] — Supprime des messages
  • [prefix]lock — Verrouille le salon
  • [prefix]unlock — Déverrouille le salon
  • [prefix]slowmode [durée] — Active le mode lent
  • [prefix]lockall — Verrouille tous les salons
  • [prefix]unlockall — Déverrouille tous les salons
  • [prefix]vider-salon — Supprime et recrée le salon

🤖 Auto-react / Message

  • [prefix]createcommande [message] — Crée une commande auto
  • [prefix]delcommande [message] — Supprime une commande auto
  • [prefix]listecommandes — Liste les commandes auto
  • [prefix]autorespond [message] — Crée une auto-réponse
  • [prefix]delrespond [numero] — Supprime une auto-réponse
  • [prefix]listrespond — Liste les auto-réponses
  • [prefix]addreact [mot] [emoji] — Ajoute une auto-réaction
  • [prefix]delreact [numero] — Supprime une auto-réaction
  • [prefix]listreact — Liste les auto-réactions

❓ Aide

  • [prefix]help — Affiche la liste des commandes

📊 Sondages

  • [prefix]poll <question> ; <option1> ; <option2> ... — Crée un sondage
  • [prefix]closepoll <ID> — Ferme un sondage

🌐 Langues

  • [prefix]setlangues — Permet de choisir ou retirer ses rôles de langues
